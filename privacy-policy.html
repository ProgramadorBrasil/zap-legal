3. Permissoes do Dispositivo

O app solicita as seguintes permissoes do Android, cada uma com finalidade especifica:

  • Contatos (READ_CONTACTS): Exibir sua lista de contatos para selecionar o destinatario da mensagem agendada.
  • Notificacoes (POST_NOTIFICATIONS): Mostrar alertas quando mensagens sao enviadas ou falham.
  • Servico de Acessibilidade: Tocar automaticamente o botao "Enviar" no WhatsApp apos a mensagem ser inserida. Este servico age SOMENTE quando voce agenda uma mensagem e funciona APENAS no WhatsApp e WhatsApp Business.
  • Alarmes exatos (SCHEDULE_EXACT_ALARM): Garantir que as mensagens sejam enviadas no horario exato agendado.
  • Servico em primeiro plano (FOREGROUND_SERVICE): Manter o servico de envio ativo durante o envio da mensagem.
  • Boot (RECEIVE_BOOT_COMPLETED): Reconfigurar alarmes agendados apos o dispositivo reiniciar.

4. Compartilhamento de Dados

Nos NAO compartilhamos, vendemos ou transferimos seus dados pessoais para terceiros.

A unica excecao sao os dados de compra processados pelo Google Play Billing, que sao gerenciados diretamente pelo Google conforme a Politica de Privacidade do Google.

5. Retencao de Dados

Seus dados sao retidos localmente no dispositivo enquanto o app estiver instalado. Voce pode deletar seus dados a qualquer momento em: Configuracoes > Privacidade > Deletar meus dados.

Ao desinstalar o app, todos os dados locais sao permanentemente removidos pelo sistema Android.

6. Seus Direitos (LGPD Art. 18)

Conforme a LGPD, voce tem os seguintes direitos:

  • Confirmacao e acesso: Confirmar a existencia de tratamento e acessar seus dados.
  • Correcao: Corrigir dados incompletos, inexatos ou desatualizados.
  • Anonimizacao, bloqueio ou eliminacao: Solicitar a exclusao de dados desnecessarios ou excessivos.
  • Portabilidade: Solicitar a portabilidade dos seus dados (exportacao de logs disponivel no app).
  • Eliminacao: Solicitar a exclusao total de dados pessoais tratados com consentimento.
  • Revogacao do consentimento: Revogar o consentimento a qualquer momento.

Para exercer qualquer destes direitos, use a opcao "Deletar meus dados" no app ou entre em contato pelo e-mail: zap.scheduler.app@gmail.com.

7. Seguranca

Implementamos as seguintes medidas de seguranca:

  • Todos os dados sao armazenados localmente no dispositivo, protegidos pelo sandbox do Android.
  • O banco de dados utiliza Room (SQLite) com acesso restrito ao app.
  • As preferencias sao armazenadas via DataStore criptografado pelo sistema.
  • Nenhum dado e transmitido pela internet (exceto dados de compra via Google Play).
  • O codigo e ofuscado via R8/ProGuard na versao release.

8. Menores de Idade

O ZAP Scheduler nao e direcionado a menores de 13 anos. Nao coletamos intencionalmente dados de criancas. Se voce acredita que uma crianca forneceu dados pessoais, entre em contato conosco para que possamos tomar as providencias necessarias.

Podemos atualizar esta Politica de Privacidade periodicamente. Alteracoes significativas serao notificadas por meio do app. A data da ultima atualizacao esta indicada no topo deste documento.
